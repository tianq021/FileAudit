# FileAudit 发布说明

FileAudit 是一个 Windows 桌面文件体检工具。它可以扫描文件夹或磁盘目录，帮你查看文件数量、目录占用、重复文件、可疑文件、扫描错误，并能导出 CSV 数据和 HTML 图表报告。

## 下载后怎么启动

如果你拿到的是打包版，只需要确认同一个文件夹里有这两个文件：

```text
FileAudit.exe
settings.json
```

双击 `FileAudit.exe` 就能启动，不需要安装 Python，不需要安装 PySide6，也不需要打开命令行。

当前项目里已经打包好的位置是：

```text
dist\FileAudit.exe
dist\settings.json
```

可以把 `dist` 文件夹里的这两个文件一起发给别人。`settings.json` 是默认设置文件，建议和 exe 放在同一目录，方便以后保存扫描规则和默认目录。

第一次启动可能会慢几秒，这是单文件 exe 正在解包运行，属于正常现象。

## 第一次怎么用

1. 双击打开 `FileAudit.exe`。
2. 在左侧进入“扫描配置”。
3. 点击“选择目录”，选择要检查的文件夹或磁盘目录。
4. 如果目录很大，先点“预检查”，看预计文件数量、跳过数量和 Hash 候选数量。
5. 点击“开始扫描”。
6. 扫描时看底部状态栏，那里会显示当前阶段、文件数、目录数、跳过数、错误数和当前路径。
7. 扫描完成后查看“扫描概览”“文件明细”“重复文件”“可疑文件”“扫描错误”。
8. 需要留档时，进入“分析+导出”，点击“生成报告”。

## 发朋友圈可以这样写

我做了一个 Windows 文件体检小工具 FileAudit，可以扫描文件夹，查看重复文件、可疑文件、目录占用、扫描错误，还能导出 CSV 和 HTML 图表报告。

使用方式很简单：把 `FileAudit.exe` 和 `settings.json` 放在同一个文件夹，双击 exe 就能运行，不用装 Python。打开后选择要检查的目录，先预检查，再开始扫描，最后可以导出报告。

## 适合检查什么

- 电脑里哪些目录占空间。
- 有没有重复文件。
- 有没有可疑扩展名、双扩展名伪装、脚本文件、隐藏文件、空文件、大文件。
- 哪些文件因为权限或超时没有扫到。
- 整理文件前先生成一份报告留底。

## 隐私提醒

默认情况下，界面会显示完整路径，报告也会导出完整路径。发报告给别人前，可以在“设置”里关闭“报告导出完整路径”，这样导出的 CSV 和 HTML 报告会只写文件名或目录名，减少路径泄露。

如果要扫描整盘，建议先用“预检查”，并在“设置”里配置跳过目录、跳过扩展名或跳过路径关键词。

## 开发者启动方式

如果你拿到的是源码，而不是打包好的 exe，当前机器不要默认使用项目里的 `.venv`。已验证可用的是 C 盘 Python 3.12：

```bash
C:\Python312\python.exe main.py
```

该环境已确认安装：

```text
PySide6 6.11.1
PyInstaller 6.20.0
```

如果要重新打包 exe，可以执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\build_exe_python312.ps1
```

生成结果在：

```text
dist\FileAudit.exe
```
