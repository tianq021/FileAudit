# 更新记录

## 2026-06-11

- 新增真实目录扫描核心，支持递归扫描、忽略目录、文件基础信息收集和错误记录。
- 新增扫描数据模型：`ScanOptions`、`FileRecord`、`DuplicateGroup`、`ScanSummary`、`ScanResult`。
- 新增后台扫描线程 `ScanWorker`，扫描过程不阻塞界面。
- 新增取消扫描功能，取消后保留已经扫描到的部分结果。
- 新增重复文件检测：先按文件大小分组，再对同大小文件计算 Hash。
- 新增风险检测：可疑扩展名、双扩展名、隐藏文件、空文件、大文件、时间异常和路径过长。
- 调整默认风险规则：`.js` 不再默认作为可疑扩展名。
- 新增扩展名白名单，白名单扩展名不触发可疑扩展名和双扩展名风险。
- 文件明细页接入真实扫描结果，支持搜索、风险筛选、文件类型筛选和表头排序。
- 重复文件页接入真实重复文件结果，显示完整路径、Hash 和可节省空间。
- 可疑文件页接入真实风险结果，并按高风险、中风险、低风险排序。
- 扫描概览页新增统计卡片和图表：文件类型分布、风险等级分布、目录占用 Top 10。
- 优化长路径显示，表格支持横向滚动并提供 tooltip。
- 优化滚动条样式，提高横向/纵向滚动条可见性。
- 新增 CSV 数据包和 HTML 图表报告导出。
- 新增设置页，支持默认目录、报告目录、阈值、Hash 算法、检测开关、忽略目录、可疑扩展名和白名单扩展名。
- 新增设置持久化，保存到 `%USERPROFILE%\.fileaudit\settings.json`。
- 修正 `.gitignore`，只忽略根目录 `/reports/`，避免误忽略源码目录 `fileaudit/reports/`。
