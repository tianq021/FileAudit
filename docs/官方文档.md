# PySide6 官方文档入口

这份文件只放官方资料，方便以后改 FileAudit 时查。

## 先看这些

1. [Qt for Python 首页](https://doc.qt.io/qtforpython-6/)

   PySide6 官方文档总入口。

2. [Qt for Python Tutorials](https://doc.qt.io/qtforpython-6/tutorials/index.html)

   官方教程入口。适合学习窗口、按钮、布局和信号槽。

3. [PySide6.QtWidgets 模块](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/index.html)

   本项目主要用的模块。`QLabel`、`QPushButton`、`QLineEdit`、`QTableWidget`、`QFileDialog`、`QScrollArea`、`QTabWidget` 都在这里。

## 和本项目最相关的页面

### 控件

- [QLabel](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QLabel.html)：显示文字。
- [QPushButton](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QPushButton.html)：按钮。
- [QLineEdit](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QLineEdit.html)：单行输入框。
- [QTextEdit](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QTextEdit.html)：多行文本框，设置页的目录、扩展名和关键词列表会用到。
- [QCheckBox](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QCheckBox.html)：检测项、导出完整路径开关。
- [QRadioButton](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QRadioButton.html)：扫描范围选择，例如“全部扫描”和“只扫描匹配规则”。
- [QComboBox](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QComboBox.html)：Hash 算法、筛选条件、隐藏文件处理和大文件处理下拉框。
- [QTabWidget](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QTabWidget.html)：扫描概览中的图表标签页。
- [QTableWidget](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QTableWidget.html)：文件明细、重复文件、可疑文件和扫描错误表格。
- [QHeaderView](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QHeaderView.html)：表格列宽、表头点击和排序指示。
- [QFileDialog](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QFileDialog.html)：选择扫描目录和报告输出目录。
- [QMessageBox](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QMessageBox.html)：错误、保存成功、导出成功提示。
- [QScrollArea](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QScrollArea.html)：设置页、扫描配置页和图表区域滚动。

### 输入校验

- [QIntValidator](https://doc.qt.io/qtforpython-6/PySide6/QtGui/QIntValidator.html)：限制数字输入。本项目用于大文件阈值、路径长度阈值、单文件超时和修改时间分类月份。

### 布局

- [QVBoxLayout](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QVBoxLayout.html)：从上到下排列控件。
- [QHBoxLayout](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QHBoxLayout.html)：从左到右排列控件。
- [QGridLayout](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QGridLayout.html)：网格排列控件。

### 事件和线程

- [Signals and Slots](https://doc.qt.io/qtforpython-6/tutorials/basictutorial/signals_and_slots.html)：按钮点击、后台线程通知 UI。
- [QThread](https://doc.qt.io/qtforpython-6/PySide6/QtCore/QThread.html)：后台扫描线程。本项目 `fileaudit/services/scan_worker.py` 使用它。

### 绘图和样式

- [QPainter](https://doc.qt.io/qtforpython-6/PySide6/QtGui/QPainter.html)：自绘图表。本项目 `BarChart` 和 `DonutChart` 使用它画图。
- [QColor](https://doc.qt.io/qtforpython-6/PySide6/QtGui/QColor.html)：图表颜色。
- [QPen](https://doc.qt.io/qtforpython-6/PySide6/QtGui/QPen.html)：图表描边。
- [QFont](https://doc.qt.io/qtforpython-6/PySide6/QtGui/QFont.html)：图表文字样式。
- [Qt Style Sheets Reference](https://doc.qt.io/qt-6/stylesheet-reference.html)：QSS 样式参考。本项目 `fileaudit/ui/styles.py` 里的颜色、按钮、标签页、表格和滚动条样式都属于这类内容。

## 建议学习顺序

1. 先看 `QLabel`、`QPushButton`，能看懂文字和按钮。
2. 再看 `QVBoxLayout`、`QHBoxLayout`、`QGridLayout`，能看懂控件怎么摆放。
3. 再看 `Signals and Slots`，能看懂点击按钮怎么触发函数。
4. 再看 `QThread`，理解为什么扫描不能直接跑在 UI 线程。
5. 再看 `QTableWidget` 和 `QHeaderView`，理解结果表格。
6. 再看 `QTabWidget`，理解概览页多个图表区域。
7. 最后看 `QPainter` 和 QSS，调整图表和样式。
