# 修改指南

这份说明只解决一个问题：你想自己改东西时，应该打开哪个文件。

## 先记住这些地方

```text
main.py
```

程序入口。一般不用改，除非你要改启动方式。

```text
fileaudit/app.py
```

主窗口。负责页面组装、启动扫描、取消扫描、分发扫描结果、导出报告和同步设置。

```text
fileaudit/ui/pages.py
```

具体页面。扫描配置、扫描概览、文件明细、重复文件、可疑文件、报告导出、设置页都在这里。

```text
fileaudit/ui/components.py
```

通用组件。顶部栏、左侧导航、底部状态栏、统计卡片和图表控件在这里。

```text
fileaudit/ui/styles.py
```

界面样式。颜色、按钮、表格、滚动条、进度条等 QSS 都在这里。

```text
fileaudit/core/scanner.py
```

扫描核心。目录遍历、风险检测、Hash 和重复文件判断在这里。

```text
fileaudit/models/scan.py
```

数据结构。扫描配置、文件记录、重复文件组、扫描结果都在这里。

```text
fileaudit/reports/exporter.py
```

报告导出。CSV 和 HTML 报告生成逻辑在这里。

```text
fileaudit/config/settings.py
```

设置读写。默认设置、JSON 保存和加载在这里。

## 常见修改怎么找

### 改软件标题

打开：

```text
fileaudit/app.py
fileaudit/ui/components.py
```

搜索：

```text
文件分析器
FileAudit 文件体检分析器
```

一个是窗口标题，一个是顶部栏标题。

### 改左侧菜单文字

打开：

```text
fileaudit/ui/components.py
```

搜索：

```python
buttons = [
    ("扫描配置", 0),
    ("扫描概览", 1),
    ...
]
```

只改双引号里的中文即可。后面的数字是页面编号，改页面顺序时才需要同步调整。

### 改扫描配置页

打开：

```text
fileaudit/ui/pages.py
```

搜索：

```python
class ScanConfigPage
```

这里可以改：

- 扫描目录输入框。
- 检测选项复选框。
- 大文件阈值。
- 路径长度阈值。
- Hash 算法。
- 忽略目录。
- 可疑扩展名。
- 白名单扩展名。

### 改扫描规则

打开：

```text
fileaudit/core/scanner.py
fileaudit/models/scan.py
```

常见位置：

```python
ScanOptions.suspicious_extensions
```

默认可疑扩展名列表在这里。

```python
_apply_risk_rules
```

风险规则在这里。比如要改隐藏文件、大文件、双扩展名的判断逻辑，就看这个函数。

### 改重复文件判断

打开：

```text
fileaudit/core/scanner.py
```

搜索：

```python
_find_duplicates
```

当前逻辑是先按大小分组，再对同大小文件计算 Hash。

### 改文件明细表格

打开：

```text
fileaudit/ui/pages.py
```

搜索：

```python
class FileDetailPage
```

这里可以改：

- 表头。
- 表格列宽。
- 搜索和筛选。
- 表头排序规则。

排序规则在：

```python
file_detail_sort_key
```

### 改重复文件页或可疑文件页

打开：

```text
fileaudit/ui/pages.py
```

搜索：

```python
class DuplicatePage
class RiskPage
```

这里负责把扫描结果填进表格。

### 改扫描概览图表

打开：

```text
fileaudit/ui/pages.py
fileaudit/ui/components.py
```

`OverviewPage` 负责把数据传给图表。

`BarChart` 负责绘制图表。

如果只改颜色和滚动条样式，打开：

```text
fileaudit/ui/styles.py
```

### 改报告导出

打开：

```text
fileaudit/reports/exporter.py
```

这里可以改：

- CSV 文件名。
- CSV 表头。
- HTML 报告样式。
- HTML 图表。
- 风险文件预览。

### 改设置页

打开：

```text
fileaudit/ui/pages.py
fileaudit/config/settings.py
fileaudit/app.py
```

对应关系：

- `SettingsPage`：设置页界面。
- `AppSettings`：设置字段。
- `load_settings` / `save_settings`：JSON 读写。
- `MainWindow.apply_settings_to_pages`：启动时把设置同步到页面。

### 改滚动条

打开：

```text
fileaudit/ui/styles.py
```

搜索：

```css
QScrollBar
```

横向和纵向滚动条都在这里。

## 最容易改错的地方

- 不要让扫描核心 `fileaudit/core/` 直接引用 PySide6 控件。
- UI 更新要在主线程做，耗时扫描要走 `ScanWorker`。
- 左侧菜单编号要和 `app.py` 里页面添加顺序一致。
- 表格排序不要再自定义 `QTableWidgetItem.__lt__`，之前会触发 PySide6 原生崩溃。
- `.gitignore` 里不要写裸 `reports/`，否则会误忽略 `fileaudit/reports/` 源码目录。
- 中文文件保持 UTF-8 编码。

## 推荐修改顺序

如果你刚开始自己改，建议按这个顺序练：

1. 先改页面文字。
2. 再改按钮文字。
3. 再改颜色、滚动条和字体大小。
4. 再改表格列。
5. 再改扫描规则。
6. 最后再改报告导出和设置持久化。
